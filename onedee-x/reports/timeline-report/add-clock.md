# How to Add Clock

## Step to Add Clock

* Go to **Reports Menu**
* Click **Timeline**

![](../../../.gitbook/assets/screenshot_4.png)

* Click **Add Clock**

![](../../../.gitbook/assets/screenshot_5.png)

* Select **Clock Type**
* Select **Zone**
* Select **Date** 
* Select **Time** 
* Enter **Reason** 
* Select **Employee** 
* Click **Done** 

![](../../../.gitbook/assets/screenshot_6.png)
