## How to talk with Bot for Clock in

* Go to "**Chat Bot OneDee**"
* Type message **"Clock in"**
* Select **Zone type**

## How to talk with Bot for Clock out

* Go to "**Chat Bot OneDee**"
* Type message **"Clock out"**
* Select **Zone type**
